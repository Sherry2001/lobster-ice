# lobster-ice
STEP 2020 Capstone Project :) 

### Step 1: Start Server
- create a file in this directory name'.env' for credential setup. (reach out for details)
- cd server
- npm install
- npm start 

### Chrome Extension 
- Navigate to chrome://extensions
- Select Load Unpacked 
- Select the chrome-extension folder

### Web Application 
- cd webapp-client
- npm install
- npm start

### Testing 
- cd server
- npm install
- npm test
